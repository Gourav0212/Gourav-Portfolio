# Gourav Portfolio

Personal portfolio website for **Gourav Dutt Shrimali** — projects, skills, contact, and DSA progress.

## Live Site (GitHub Pages)

This repo includes a GitHub Actions workflow that deploys the site to **GitHub Pages** on every push to `main` (see `.github/workflows/static.yml`).

- Expected URL: `https://gourav0212.github.io/Gourav-Portfolio/`
- If it’s not live yet: go to **Repo Settings → Pages** and set the source to **GitHub Actions**.

## Tech Stack

- HTML, CSS, JavaScript
- Tailwind CSS (CDN)
- Font Awesome (CDN) + Google Fonts (Poppins)

## Local Setup

Option 1 (simple):
- Open `index.html` in your browser

Option 2 (recommended):
- Use VS Code **Live Server** (or any static server) and open `index.html`

## Project Structure

- `index.html` — page layout/sections
- `style.css` — custom styling/theme
- `jscode.js` — data + dynamic rendering (skills/projects/typewriter/DSA)
- `favicon/` — site favicon

## Customize

- Update your skills/projects: `jscode.js`
- Update layout/sections/content: `index.html`
- Update colors/animations/styles: `style.css`
